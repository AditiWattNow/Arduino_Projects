# Potentiometer LED Dimmer

## Description
This project adjusts the brightness of an LED using a potentiometer by reading analog values and applying PWM.

## Components Used
- Arduino UNO
- 1 LED
- 330Ω Resistor
- 10kΩ Potentiometer
- Breadboard and jumper wires

## Circuit Diagram
https://www.tinkercad.com/things/dnksGsvfR6J-neat-wluff/editel?returnTo=https%3A%2F%2Fwww.tinkercad.com%2Fdashboard

## Code Explanation
- Reads analog value from potentiometer.
- Adjusts LED brightness via PWM on a digital pin.

## How to Run
1. Assemble the circuit.
2. Upload `Potentiometer_LED_Dimmer.ino` to Arduino.
3. Turn the potentiometer to control LED brightness.

## Output
LED brightness varies smoothly as potentiometer is rotated.
