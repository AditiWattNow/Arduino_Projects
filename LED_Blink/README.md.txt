# LED Traffic Light Blink

## Description
This Arduino project blinks three LEDs (Red, Yellow, Green) one after the other , yellow for 5 times, red for 10 times and green for 15 times.

## Components Used
- Arduino UNO
- Red, Yellow, Green LEDs
- 330Ω Resistors (3x)
- Breadboard and jumper wires

## Circuit Diagram
https://wokwi.com/projects/433355316424996865

## Code Explanation
- Pin connections:
  - Red LED: Pin 12
  - Yellow LED: Pin 13
  - Green LED: Pin 7

## How to Run
1. Build the circuit as per diagram.
2. Upload `LED_Blink.ino to Arduino.
3. Observe LEDs blinking in sequence.

## Output
 Yellow → Red → Green → Repeat
